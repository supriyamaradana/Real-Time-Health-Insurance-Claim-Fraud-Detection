# Real-Time Health Insurance Claim Fraud Detection

## Project Overview
This project uses Machine Learning and Power BI to detect fraudulent healthcare insurance claims and provide real-time fraud analytics through interactive dashboards.

## Technologies Used
- Python
- Jupyter Notebook
- XGBoost
- Pandas
- NumPy
- Scikit-learn
- Power BI

## Features
- Fraud Detection using XGBoost
- Healthcare Claim Analysis
- Interactive Power BI Dashboards
- Fraud Trend Monitoring
- Diagnosis and Procedure Analysis
- AI-Based Fraud Insights

## Machine Learning Model
Algorithm Used:
- XGBoost Classifier

Model Accuracy:
- 65.29%

## Dashboard Pages
1. Executive Dashboard
2. Fraud Analytics Dashboard
3. AI & Machine Learning Insights

## Project Files
- Jupyter Notebook (.ipynb)
- Power BI Dashboard (.pbix)
